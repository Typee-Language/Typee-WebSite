- Typee Language Lessons -

Typee language is a new generic Object Oriented Programming language. The syntax of this programming language is somewhat new while it has been largely inspired by OOP languages such as C++11, Java 8 or Python 3.7. To be able to efficiently program with Typee, the characteristics of the language have to be learned. So, here is the place to learn programming with Typee but... this documentation is currently under writing. Never mind. You will soon get here the underlying concepts of the language. You will be explained its grammar as well as its syntax. You will learn how to use it efficiently, not only about programming in Typee or about implementing algorithms in Typee, but also about how to translate your Typee programs in Python 3, Java 8 or C++11 and how your Typee source code interfaces with these targetted foreign programming languages.

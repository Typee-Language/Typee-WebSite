- Typee Documentation -

You will find here the whole documentation of language Typee. Up today it is still under creation since this Web site has been created on Aug. 1st, 2018, but very first documentation pages are very soon to come. In the meantime, do not hesitate to have a look at Typee repository. You will get there a full access to the current status of the Typee Project development. There, documentation is also available, either directly as Mark Down documents (mainly, readme.md files) or as HTML documentation of the already developed code for Typee language implementation. Up today available sections are:
1. Documentation Introduction 2. Typee Grammar Specifications 3. Typee Language Full Description (currently on progress) Annex A. A State of the Art on Translation of Computer Languages
Please remember that this site is currently under construction. As soon as section 3. will have been completed, we will invite you to not hesitate to go straight to it, to get a deep inside view of the language characteristics. A section 4. (still to come) will provide also a deep dive into built-in libraries programmed in Typee. [caption id="attachment_46" align="alignright" width="563"] (photo credit: Keli Clark)[/caption]
